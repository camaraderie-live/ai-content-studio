# 品牌檔案部：客戶元件庫

元件庫是客戶的品牌資料庫。**第一準則：產出必須是「客戶專屬的內容」而不是「AI 生成的東西」**——一切生成都以元件庫為根據。

## 元件庫放哪裡

**放在新手包專案根目錄的 `我的品牌/` 資料夾**。這個資料夾已加進 `.gitignore`，客戶個資永遠不會被 commit 或推上 git；`範例/` 是給客戶看的範本，**任何客戶的真實資料都不准存進 `範例/`**。第一次建檔時直接幫客戶 `mkdir` 建好整個結構：

```
我的品牌/
├── 我是誰.md      # 定位、受眾、產品、禁忌話題
├── 我怎麼說話.md  # 口氣、常用詞、絕不用的詞、句式習慣
├── IG截圖/        # IG 版面與貼文截圖（視覺風格參考）
├── 待使用腳本/    # 自己寫、尚未發布的腳本與內容思路
├── 舊貼文/        # 自己發過的內容：舊貼文、舊影片轉錄稿、已發布成品歸檔
├── 對標/          # 別人的、拿來學習的：對標 Reels 轉錄稿、成功案例
└── 成品/          # 做好但還沒發布的交付物
    ├── FB/
    ├── IG/
    ├── Threads/
    └── 小紅書/
```

檔案與資料夾一律**中文命名**（IG、Reels、FB、Threads 這種平台名例外）。

## 檔案去向對照表（唯一的家，存檔前查這張表）

| 這是什麼 | 存哪裡 |
|---|---|
| 客戶自己寫、尚未發布，想先保留思路的腳本 | `待使用腳本/` |
| 客戶自己的舊貼文、舊影片轉錄稿 | `舊貼文/` |
| 客戶已發布的成品（歸檔） | `舊貼文/` |
| 別人的 Reels 轉錄稿、對標範本、成功案例 | `對標/` |
| 剛做好、還沒發布的交付物 | `成品/<平台>/` |
| IG 版面截圖 | `IG截圖/` |
| 品牌定位、受眾描述 | `我是誰.md` |
| 口氣、用詞規則 | `我怎麼說話.md` |
| 選好還沒寫（或已寫）的題目清單 | `選題庫.md` |

分不清「自己的還是別人的」就直接問客戶一句，不要猜。

分不清「已經發布還是尚未發布」也要直接問：尚未發布的本人腳本進 `待使用腳本/`；已發布內容才進 `舊貼文/`。`成品/<平台>/` 保留給 AI 剛完成、尚未發布的正式交付稿，不要和使用者自己帶來保存思路的腳本混在一起。

## 訪談建檔（第一次建元件庫）

用 grill-me 的方式：**一次一題、每題附建議答案方向**，問完一題存一題。問到哪一項，就把 `範例/` 裡對應的範例秀給客戶看（「像這樣寫就可以了」），降低他的心理門檻。

必問題目（依序）：

1. 你是誰、在做什麼？（→ 我是誰.md：一句話定位）
2. 內容是給誰看的？（→ 我是誰.md：受眾）
3. 希望粉絲看完有什麼感覺或行動？（→ 我是誰.md）
4. 有沒有絕對不碰的話題或不用的詞？（→ 我怎麼說話.md：禁忌）
5. 丟 2~3 篇你自己寫過、覺得「這就是我」的貼文給我（→ 舊貼文/，並從中萃取口氣寫進 我怎麼說話.md，寫完唸給客戶確認）
6. 有沒有你欣賞、想學習結構的帳號或貼文？（→ 對標/）
7. IG 版面截圖幾張（→ IG截圖/）

我怎麼說話.md 不是客戶自己寫的，是你從他的舊貼文萃取出來、他確認過的。

## 守門規則（生成前必查）

**動筆生成任何內容前：**

1. 讀 `我是誰.md` + `我怎麼說話.md`。不存在 → 先走訪談建檔。
2. 檢查這次任務需要的資訊夠不夠（主題、平台、目的、素材）。
3. 不夠 → **一次一題問到齊，不齊不生成**。寧可多問一題，不要生出「AI 味的東西」。
4. **硬規則：`範例/` 不是資料來源**——它只能在訪談時當「檔案長這樣」的格式示意秀給客戶看。幫客戶思考、生成、判斷時一律只根據 `我的品牌/`；`我的品牌/` 是空的就當作零資料，走訪談問到齊，不准拿 `範例/` 的內容（即使看起來跟客戶有關）來代替客戶的答案。

## 任務交付後（每次都要）

1. 交付物先存 `成品/<平台>/`。
2. **主動問一句：「這次的成果要不要存進元件庫？」**——好的產出存回去，元件庫會越用越準。
3. 客戶說已經發布了 → 問「要不要把它歸檔進 舊貼文/？」歸檔後它就成為未來 我怎麼說話.md 的參考來源。
4. `待使用腳本/` 裡的內容發布後 → 問「要不要移到 舊貼文/ 歸檔？」；使用者同意後再移動，讓內容狀態保持正確。

## 隱私

- 元件庫 `我的品牌/` 已在 `.gitignore`，不會被 commit。如果客戶想改資料夾名稱，改完要同步更新 `.gitignore`，確認新名稱一樣被擋住。
- `範例/` 會被 commit，所以它只能放範本內容；客戶的東西一律進 `我的品牌/`。
