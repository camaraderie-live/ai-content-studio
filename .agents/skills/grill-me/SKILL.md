# 打破砂鍋問到底：把使用者的影片構思問清楚

當使用者想剪的東西**沒有現成範例可以套**，或他自己也講不太清楚想要什麼時，不要急著開剪、也不要自己腦補。用一次一題的方式，把他腦中的畫面一層一層挖出來，問到你和他對同一件事為止。

**全程繁體中文，語氣輕鬆，一次只問一題。**

## 什麼時候套用

- `aroll-guide` 裡三種常見主線都對不上使用者的需求。
- 使用者說「我不知道怎麼形容」「你幫我想」「我有個感覺但講不出來」。
- 使用者給的方向太模糊（例如「剪好看一點」「有質感」），沒辦法直接變成剪輯策略。
- 使用者講得出主題（例如「介紹我們的產品」），但長度、平台、想要的感覺大多還沒講清楚時。

## 怎麼問

1. **一次只問一題。** 問完一題、等他回答，再問下一題。不要一次丟五個問題把人淹掉。
2. **每一題都附上你的建議答案。** 不要只丟開放問題讓他從零想——給他一個預設選項（「我猜你想要 A，這樣對嗎？」），他點頭或修正都比憑空發想輕鬆。
3. **從大到小，一層一層往下。** 先問整支片的目的和感覺，再問主線、素材、細節。上一題的答案會決定下一題要問什麼。
4. **能自己看的就不要問。** 素材資料夾裡看得到的東西（有幾個檔、多長、叫什麼），自己先 `ls` / `ffprobe` 看過，不要拿這種問題浪費使用者。

## 建議的提問順序（依情況調整）

由粗到細，走完一條決策樹：

1. **這支片給誰看、想達到什麼？**（賣產品／自我介紹／教學／純記錄）——這決定整體調性。
2. **想要什麼感覺？** 給他幾個具體選項對比，而不是問「你想要什麼風格」：快節奏 IG 短影音感／沉穩訪談感／溫馨 Vlog 感／乾淨教學感。
3. **主線是什麼？**（對照 `aroll-guide`：有人講話？先錄旁白配畫面？還是純畫面？）
4. **手上有哪些素材？** 缺什麼？——這裡把缺的 B-roll、截圖、參考風格列成清單請他補。
5. **有沒有非放不可的重點？**（一定要出現的一句話、一個畫面、一個 call to action）
6. **長度、平台？**（30 秒 / 1 分鐘 / 3 分鐘；直式 IG / 橫式 YouTube）

## 問完之後

把你聽到的，用自己的話**複述成一個完整的剪輯構想**再念給他聽（「所以你想要的是：一支 60 秒、直式、快節奏的產品介紹，主線是你的口播，中間疊三張截圖……對嗎？」）。他說「對」了，才接著進到正式的剪輯策略（video-use SKILL.md Hard Rule 11：提策略、等核准、才動手）。

如果複述完他又補了新東西，就再繞回上面繼續問，直到真的對齊。**問清楚的成本，遠低於剪錯重來。**
