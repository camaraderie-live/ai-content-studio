# 使用指南（從零開始的路線圖）

> 這份文件幫你做一件事：告訴你該照什麼順序看哪份教學。每份教學只教一件事，照順序走就好。

1. [安裝 Claude Desktop](./教學/01-安裝%20Claude%20Desktop.md) —— 還沒裝 App 的第一步
2. [打開這個專案](./教學/02-打開這個專案.md) —— 讓 Claude 開始為你工作、自動裝好工具
3. [申請轉錄金鑰](./教學/03-申請轉錄金鑰.md) —— 第一次使用會被要求貼一把免費金鑰
4. [剪出第一支影片](./教學/06-剪出第一支影片.md) —— 放素材 → 說需求 → 拿成品
5. [把專案更新到最新版](./教學/04-更新專案到最新版.md) —— 之後有新功能時用
6. [卡住了怎麼辦](./教學/05-卡住了怎麼辦.md) —— 遇到任何問題先看這份

> 目前僅支援 Mac。Windows 使用者可以直接問 Claude 怎麼做到類似的事，但自動安裝流程不保證能用。
