# 使用指南（圖文步驟）

> 這份指南裡每個步驟都有一個「請在這裡插入截圖」的位置。實際操作一次、把對應截圖存進 `screenshots/` 資料夾、檔名對上下面標的編號，圖片就會自動顯示出來。

## 使用前需知
- 這個專案在 windows 系統上面操作可能會遇到問題，如果有遇到問題可以先問 claude

## 步驟 1：安裝 Claude Desktop（如果還沒裝）

前往 https://claude.ai/download 下載 Mac 版並安裝，用你的帳號登入。

`![步驟1：Claude Desktop 下載頁面](screenshots/01-download.png)`
<!-- 請在這裡插入截圖：claude.ai/download 的畫面，或安裝完成後的登入畫面 -->

## 步驟 2：開啟這個資料夾

打開 Claude Desktop，找到「開啟資料夾」或「新增 Project」的入口，選擇你解壓縮後的「AI 剪片新手包」資料夾。

`![步驟2：開啟資料夾入口](screenshots/02-open-folder.png)`
<!-- 請在這裡插入截圖：Claude Desktop 裡點擊開啟資料夾/新增 Project 的按鈕位置 -->

`![步驟2：選擇新手包資料夾](screenshots/03-select-folder.png)`
<!-- 請在這裡插入截圖：Finder 選擇資料夾的視窗，選到 ai-editing-starter-kit -->

## 步驟 3：跟它打聲招呼，讓它自動檢查環境

資料夾打開後會進到一個對話畫面。隨便打個招呼（例如「你好」），Claude 會自動開始檢查電腦裡有沒有裝好剪片需要的工具，缺什麼會自動幫你裝，這個過程可能需要幾分鐘，跟裝 App 一樣正常。

`![步驟3：Claude 自動檢查環境的畫面](screenshots/04-env-check.png)`
<!-- 請在這裡插入截圖：Claude 回報環境檢查清單/自動安裝進度的對話畫面 -->

## 步驟 4：貼上轉錄金鑰（第一次使用才需要）

Claude 會請你去 https://elevenlabs.io/app/settings/api-keys 註冊一組免費的 API 金鑰（用來把影片裡講的話轉成文字），複製後貼到對話框裡給它。

`![步驟4：ElevenLabs 申請金鑰頁面](screenshots/05-elevenlabs-key.png)`
<!-- 請在這裡插入截圖：ElevenLabs 網站上顯示/複製 API key 的位置 -->

## 步驟 5：看到「環境就緒」的回覆

Claude 會跟你說環境都設定好了，可以開始丟影片進來了。

`![步驟5：環境就緒的回覆](screenshots/06-ready.png)`
<!-- 請在這裡插入截圖：Claude 回報「環境全部就緒」的對話畫面 -->

## 步驟 6：建立影片專案資料夾，放好素材

打開 Finder，在「AI 剪片新手包」**外面**（例如「影片」資料夾裡）另外建一個專案資料夾，裡面再建兩個子資料夾：

- `A-ROLL/`：主影片、主聲道（口播、訪談這種主線畫面）
- `B-ROLL/`：要疊加上去的素材（截圖、輔助畫面、參考圖片）

素材檔名用**數字開頭依序命名**（`1. 開場口播.mp4`、`2. 洞察報告截圖.png`），順序一目了然。想做客製化動畫效果的話，最好一併提供 `.svg` 向量圖放進 `B-ROLL/`。

> 不想自己建也可以：跟 Claude 說「幫我建一個影片專案資料夾」，它會問你要放哪、叫什麼名字，然後幫你建好 `A-ROLL/`、`B-ROLL/`。

`![步驟6：影片專案資料夾結構](screenshots/07-drag-video.png)`
<!-- 請在這裡插入截圖：Finder 視窗，專案資料夾底下有 A-ROLL、B-ROLL 兩個子資料夾的樣子 -->

## 步驟 7：跟 Claude 說想剪成什麼樣子

回到對話框，直接說出你想要的結果，例如「幫我把這幾支影片剪成一支 3 分鐘的重點精華，加中文字幕」。不知道怎麼講可以參考 [常用句型.md](./常用句型.md)。

`![步驟7：提出剪輯需求](screenshots/08-request.png)`
<!-- 請在這裡插入截圖：使用者輸入剪片需求、Claude 開始分析素材的對話畫面 -->

## 步驟 8：確認剪輯策略

Claude 會先用白話文跟你說明它打算怎麼剪（哪些片段留、怎麼配色、字幕風格等），等你說「可以」或提出調整，它才會真的開始剪。

`![步驟8：確認剪輯策略](screenshots/09-confirm-strategy.png)`
<!-- 請在這裡插入截圖：Claude 提出剪輯策略、等待確認的對話畫面 -->

## 步驟 9：預覽成品

剪完會給你一個預覽版本，滿意的話再請它輸出正式版，檔案會出現在資料夾裡的 `edit/final.mp4`。

`![步驟9：預覽與輸出](screenshots/10-preview.png)`
<!-- 請在這裡插入截圖：Claude 給出預覽/最終檔案位置的對話畫面，或 Finder 裡 edit/final.mp4 的位置 -->

## 步驟 10：不滿意想改怎麼辦

直接用一般說話的方式告訴它要改哪裡，例如「第 2 段太快了，慢一點」或「字幕改成粉紅色」，它會照著你的意見重新調整，不用重新剪一次。
