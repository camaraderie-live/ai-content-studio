# 腳本 + 動畫標註（想要哪裡加什麼效果，直接寫在旁白旁邊）

如果你心裡已經有一段要講的話、也大概知道哪一句想配什麼畫面，最快的方式就是**把腳本一行一行寫下來，在每一句後面用括號註明你想要的畫面或動畫**。你不用會任何剪輯術語，用大白話描述就好，我會照著做——做之前我會先用我的話跟你複述一次每一條要幹嘛、有沒有缺素材，你確認「對」我才動手。

## 格式：一行旁白 +（括號寫你想要的畫面）

```
你講的那句話（括號裡寫這句話畫面上要出現什麼、怎麼動）
下一句話（同樣在括號裡描述）
```

沒有註記的句子，就維持你原本的畫面、不加東西。

## 完整範例（照這個感覺寫就好）

```
你現在看到的這個影片，完全都是用 Claude 剪的（附上 Claude 在跑步的動畫）
看到最後你也能輕鬆使用，我完全沒碰任何剪輯軟體（附上剪輯軟體的圖片在上面）

我是阿閎，學習經營自媒體一個月（外框加上 IG 自媒體短影音的感覺）
到此刻為止總觀看次數 8.4 萬、觸及帳號快四萬（附上截圖）
我並沒有使用什麼爆款鉤子、或是時事梗（飛入「爆款鉤子」圖片）
我承認那些東西很有用
但我只想完全做自己（把當前畫面放到最大 zoom in）
所以我很驚訝能有這樣的數字（畫面疊上數字從 0 暴漲到 84k 的動畫）
所以我決定在這裡記錄我學習經營自媒體的過程
也建立了一個讓大家能互相交流的社群
我本身是一個軟體工程師
我會同時分享我如何用 AI 剪片而且有不錯的成效（附上 demo-1、demo-2，左右飛入）
你不需要懂寫程式也能輕鬆上手（加上「終端機」的動畫）
你可以把時間花在
拍攝好的 B-roll（疊上好的 B-roll 畫面）
或是撰寫真正適合你的腳本
歡迎追蹤我，我會分享更多 AI 剪片的技巧
```

## 你可以「點菜」的動畫（不知道怎麼寫就照抄這些）

| 你想要的效果 | 這樣寫 | 看起來像 | 你要準備的素材 |
|---|---|---|---|
| 疊一段小動畫 / GIF | 「附上 XXX 的動畫」 | 角落或畫面上多一段會動的小圖 | 那個動畫或 GIF 檔（放 B-ROLL） |
| 疊一張圖 / 截圖 | 「附上 XXX 的圖片 / 截圖在上面」 | 畫面上蓋一張圖 | 那張圖片或截圖（放 B-ROLL） |
| 從旁邊飛進來 | 「飛入 XXX」「左右飛入」 | 圖片從左邊或右邊滑進畫面 | 要飛進來的圖（放 B-ROLL） |
| 放大聚焦 | 「zoom in」「把畫面放到最大」 | 鏡頭慢慢推近、強調當下這句 | 不用，用你原本的畫面 |
| 數字往上跳 | 「數字從 0 暴漲到 84k」 | 數字快速滾動增加的動態圖 | 不用，我用主題色做 |
| 加外框 / 版型感 | 「外框加上 IG 短影音的感覺」 | 畫面外圈加一層邊框或標題條 | 不用，我用主題色做 |
| 打字機 / 終端機感 | 「加上終端機的動畫」 | 文字像在鍵盤上一個字一個字打出來 | 不用，我用主題色做 |
| 蓋一段 B-roll | 「疊上 XXX 的 B-roll」 | 你講話的同時畫面切到輔助畫面 | 那段 B-roll 影片（放 B-ROLL） |

顏色（外框、數字、終端機那些）會自動套用你設定的**主題色**，整支影片才會一致。還沒設定主題色的話，我剪之前會先問你。

## 三個小提醒

1. **需要素材的效果，先把檔案丟進專案的 `B-ROLL/` 資料夾**（截圖、demo 影片、要飛進來的圖、Claude 動畫等）。我複述腳本時會把「缺哪些素材」列給你，你補齊我再開工。
2. **檔名用數字開頭、照出現順序命名**（例如 `1. Claude 動畫.gif`、`2. 剪輯軟體.png`、`3. 觀看數截圖.png`），這樣我對「第幾個素材」不會對錯。
3. **不確定某個效果做不做得出來，直接問我**，或先寫上去——做不到我會老實跟你說、給你替代方案，不會硬掰。
