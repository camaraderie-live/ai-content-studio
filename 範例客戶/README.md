# 範例客戶：Sean 🏀（@still.hooping.sean）

> 這份文件幫你做一件事：讓你知道元件庫每個位置該放什麼、長什麼樣。

這是一個**完整填好的元件庫範例**，用真實帳號「31 歲的籃球夢」示範。品牌檔案部訪談時，問到哪一項就把對應的檔案秀給客戶看：

| 訪談問到…… | 秀這份 |
|---|---|
| 你是誰、給誰看、想帶來什麼 | [profile.md](./profile.md) |
| 你怎麼說話（口氣萃取長怎樣） | [voice.md](./voice.md) |
| 版面截圖要截什麼 | [ig-screenshots/](./ig-screenshots/) |
| 舊貼文、轉錄稿存起來長怎樣 | [past-posts/](./past-posts/)（內含 Reels 轉錄稿範例） |
| 對標範本長怎樣 | [benchmarks/](./benchmarks/)（四種內容型態各一份） |
| 做好的成品放哪 | [成品/](./成品/)（依平台分資料夾） |

提醒：這是「範例」不是「模板」——客戶自己的元件庫放在**他自己的資料夾**（repo 外），內容照他的品牌來，不要抄這裡的。
