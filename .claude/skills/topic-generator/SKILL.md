# 選題產生器：五個維度挖出「只有你能寫」的題目

**核心原則**：好題目來自客戶本人的經歷，不是 AI 憑空想的。所以先挖元件庫、再問客戶，絕不直接生一串「大家都能寫」的通用題目。

## 流程

1. **守門（同 profile-library）**：先讀 `我的品牌/我是誰.md`、`我怎麼說話.md`；沒建檔就先走訪談。
2. **先自己挖**：把 `舊貼文/`（他寫過什麼、哪些反應好）和 `對標/`（他欣賞什麼結構）讀一輪，對照下面五個維度，先產出你挖得到的候選題目。**每個題目都要能指出「根據你的哪段經驗」**——指不出來的不准列。
3. **缺的維度再問**：哪個維度挖不到料，就用該維度的引導提問**一次一題**問客戶（附建議答案方向，grill-me 式），問到有料為止。不用五個維度都問滿，有 2~3 個維度有料就夠出一輪題目。
4. **輸出候選清單**（8~12 個），每個題目一行：
   `題目（一句話鉤子的雛形）｜維度｜根據你的什麼經驗｜適合的平台`
5. 客戶挑完 → 存進 `我的品牌/選題庫.md`（附狀態：待寫／已寫）→ 問「要不要現在就寫？」要就交棒 `content-planner`。

## 五個維度與引導提問

### 1. 訣竅——方法、步驟與眉角
拆解客戶做成過的事。問：你做到那件事的訣竅是什麼？能簡化成幾個步驟？新人最大的挑戰是什麼、怎麼解？有哪些外行看不到的眉角？只給新手一句最中肯的建議會是什麼？

### 2. 工具——問題、資源與習慣
問：做決策時你會問自己什麼問題？收穫最大的學習資源是哪個？最離不開的軟體或工具？你對它們的真實評價？哪些日常習慣造就了你的成果？

### 3. 分析——數據與熱點
問：你的領域有什麼數據或趨勢是你有看法的？發展有天花板嗎、為什麼有人停滯？最近的熱門話題你的（反面）論點是什麼？有哪些被大家忽略的趨勢？

### 4. 啟發——故事、課題與彎路
問：犯過哪個印象最深的錯？從中學到什麼課題、過程發生了什麼？這一兩年你自己最大的轉變是什麼？第一次做某件事時的挑戰與心境？

### 5. 觀點——獨特看法與價值主張
問：哪件事你的看法和多數人不同？你抱持強烈意見的事情是什麼？什麼應該被討論卻沒人在討論？大家常忘記、一犯再犯的錯是什麼？

## 硬性規則

1. **不虛構**：每個候選題目必須錨定客戶元件庫或客戶親口說的經驗；挖不到就問，不齊不出題。
2. **一次一題**：追問時一次只問一題，附建議答案方向，不要五個維度一次轟炸。
3. 同一個經驗可以從不同維度出好幾個題目（一個故事＝啟發＋觀點＋訣竅三種切法），主動示範這種一料多用。
4. 客戶說「這題不像我」→ 直接刪，並問一句為什麼，把原因記進 `我怎麼說話.md` 的禁忌或 `選題庫.md`。
