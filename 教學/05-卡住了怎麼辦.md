# 卡住了怎麼辦（常見問題）

> 這份文件幫你做一件事：遇到問題時，知道下一步該做什麼。

**大原則：把你看到的畫面或訊息直接複製貼給 Claude，跟它說「我卡在這裡」。** 它會用白話幫你排查，九成的問題這樣就能解決。

## 常見狀況

| 狀況 | 怎麼辦 |
|---|---|
| Claude 一直沒回應 | 關掉對話重開一個新對話，打「繼續剛剛的工作」 |
| 要求輸入電腦密碼 | 這是安裝工具的正常步驟，輸入你開機用的密碼（畫面上不會顯示字，輸完按 Enter） |
| 說金鑰錯誤或過期 | 照 [03-申請轉錄金鑰](./03-申請轉錄金鑰.md) 重新申請一把貼給它 |
| 影片下載不了 | 跟 Claude 說，它會換方式重試；私人帳號的影片拿不到是正常的 |
| 剪出來的影片不滿意 | 不用重來，直接說要改哪（「第 2 段太快」「字幕換黃色」） |
| 我用 Windows | 目前僅支援 Mac。可以直接問 Claude 怎麼在 Windows 做到類似的事，但自動安裝不保證能用 |
| 怎麼問都解決不了 | 開一個全新對話，把問題從頭描述一次（新對話沒有舊包袱，常常就通了） |
