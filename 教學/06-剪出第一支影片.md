# 剪出你的第一支影片

> 這份文件幫你做一件事：從放素材到拿到成品，走完第一次剪片。

## 1. 建立影片專案資料夾

在這個新手包**外面**（例如「影片」資料夾裡）建一個專案資料夾，裡面分兩個子資料夾：

- `A-ROLL/`：主影片、主聲道（口播、訪談這種主線）
- `B-ROLL/`：要疊加上去的素材（截圖、輔助畫面、參考圖）

素材檔名用**數字開頭依序命名**（`1. 開場口播.mp4`、`2. 報告截圖.png`）。想做客製動畫，最好提供 `.svg` 向量圖放進 `B-ROLL/`。

> 不想自己建？跟 Claude 說「幫我建一個影片專案資料夾」就好。

`![專案資料夾結構](../screenshots/07-drag-video.png)`

## 2. 說出你想要的結果

例如：「幫我把這幾支剪成 3 分鐘精華，加中文字幕」，並告訴它專案資料夾在哪。不知道怎麼開口就翻 [常用句型](../常用句型.md)；想一次講清楚整支影片，用 [腳本動畫標註](../腳本動畫標註.md) 的寫法。

`![提出剪輯需求](../screenshots/08-request.png)`
<!-- 請在這裡插入截圖：使用者輸入剪片需求的對話畫面 -->

## 3. 確認策略 → 等它剪

Claude 會先用白話說明打算怎麼剪，**你說「可以」它才會動手**。有意見就直接講。

`![確認剪輯策略](../screenshots/09-confirm-strategy.png)`
<!-- 請在這裡插入截圖：Claude 提出剪輯策略、等待確認的對話畫面 -->

## 4. 預覽 → 修改 → 完成

剪完會先給預覽，滿意再輸出正式版（在專案資料夾的 `edit/final.mp4`）。不滿意直接說要改哪，不用重剪。
