# 用 Claude Desktop 打開這個專案

> 這份文件幫你做一件事：讓 Claude 讀到這個資料夾，開始為你工作。

1. 打開 Claude Desktop，找到 **Code**（寫程式／開資料夾）的入口——在左側選單或首頁上。
2. 點「開啟資料夾」，選到你下載解壓縮後的這個專案資料夾。

3. 進到對話畫面後，**隨便打個招呼**（例如「你好」）。Claude 會自動開始檢查你電腦裡的工具，缺什麼就自動裝，過程幾分鐘，中間可能請你做兩件事：
   - 輸入一次電腦密碼（裝 Homebrew 時需要）
   - 貼一組免費的轉錄金鑰（看 [03-申請轉錄金鑰](./03-申請轉錄金鑰.md)）

4. 看到 Claude 說「環境全部就緒」就完成了。
