# 把專案更新到最新版

> 這份文件幫你做一件事：拿到這個專案的最新功能與修正。

最簡單的方式：**直接跟 Claude 說「幫我把專案更新到最新版」**，它會幫你執行，你不用碰任何指令。

## 想自己動手的話

第一次下載（還沒有這個資料夾）：

```
打開「終端機」App，貼上：
git clone <這個專案的網址>
```

已經有資料夾、只是要更新：

```
打開「終端機」App，貼上：
cd 你的專案資料夾路徑
git pull
```

> 你自己建立的 skill（`my-` 開頭的資料夾）和你的品牌元件庫都不會被更新動到，放心更新。

更新完重新打開 Claude 對話就會生效。卡住了看 [05-卡住了怎麼辦](./05-卡住了怎麼辦.md)。
