# AI Content Studio — 你的一人自媒體公司

有任何問題可以直接在 [POVU Line 社群發問](https://line.me/ti/g2/sY3G_OfDiuG4LRRBebDYjClU6cVi9Awo_AL5ZQ?utm_source=invitation&utm_medium=link_copy&utm_campaign=default)

把這個資料夾用 Claude 打開，你就擁有一間**專屬於你的小型 AI 創作公司**：它會先訪談你、建立你的品牌檔案，然後幫你——

- 🎬 **剪影片、上字幕、做動畫特效**（用聊天的方式，不用學剪輯軟體）
- ✍️ **寫 FB 長文、IG 輪播圖文、短影音腳本、Threads 文章**
- 🎯 **寫出來像你本人**——一切生成都以你的品牌檔案為根據，資訊不夠它會先問，不會亂編
- 📚 **越用越懂你**——好的成果存回你的元件庫，下次直接沿用

你完全不需要會寫程式。

## 需要準備什麼

- 一台 Mac（目前僅支援 Mac；Windows 的朋友可以直接問 Claude 怎麼做到類似的事，但自動安裝流程不保證能用）
- [Claude Desktop](https://claude.ai/download) App，能使用裡面的 Code 功能
- 其他工具都會在你第一次打開這個資料夾時自動檢查、自動裝好

## 第一步點哪裡

👉 打開 [使用指南.md](./使用指南.md)，照順序走就好。每份教學只教一件事，全程白話。

不知道怎麼開口就翻 [常用句型.md](./常用句型.md)，有現成句子可以照抄。

## 背後用了什麼（可以不用管）

剪片引擎是開源專案 [video-use](https://github.com/browser-use/video-use)，動畫字幕引擎是 HyperFrames，我把它們包成新手也能直接上手的版本。
