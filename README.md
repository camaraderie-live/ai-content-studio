# AI 剪片新手包

嗨，我是阿閎 👋 這是我在「阿閎的自媒體日記」裡提到的 AI 剪片新手包。

你完全不需要會寫程式。這個資料夾背後用的是開源專案 [video-use](https://github.com/browser-use/video-use)，我把它包成新手也能直接上手的版本——**用聊天的方式剪片**，不用學任何剪輯軟體的介面。

## 需要準備什麼

- 一台 Mac
- 安裝好 [Claude Desktop](https://claude.ai/download) App，並且能使用裡面的 Code / Projects 功能
- 其他工具（ffmpeg、剪片引擎、轉錄金鑰）都會在你第一次打開這個資料夾時，由 Claude 自動幫你檢查、自動幫你裝好

## 怎麼開始

1. 把這個資料夾整個下載下來，解壓縮，放在你電腦裡任何地方都可以。
2. 用 Claude Desktop 打開這個資料夾（詳細圖文步驟看 [使用指南.md](./使用指南.md)）。
3. 跟它打聲招呼，它會自己檢查環境、缺什麼裝什麼，中間可能會請你貼一次免費的轉錄 API 金鑰。
4. 環境就緒後，在**新手包資料夾外面**另外建一個影片專案資料夾，裡面分成 `A-ROLL/`（主影片、主聲道）和 `B-ROLL/`（要疊加的素材），素材檔名用數字開頭依序命名（`1. 開場.mp4`、`2. 截圖.png`）。不會自己建也沒關係，跟 Claude 說一聲它會幫你建好。放好後告訴它這個資料夾在哪、想剪成什麼樣子就行。
5. 不知道怎麼開口就翻 [常用句型.md](./常用句型.md)，裡面有現成的範例可以直接照抄修改。

## 這包東西可以做到什麼

- 自動剪掉贅字、口頭禪、卡頓的空白
- 自動調色
- 燒錄字幕（雙語、樣式都可以聊出來）
- 生成動畫效果、圖卡（進階功能，聊天時提出來就會做）
- 每次剪完都會記住這個專案的狀態，下次接著改也沒問題

## 資料夾裡有什麼

| 檔案 | 用途 |
|---|---|
| `CLAUDE.md` | Claude 打開這個資料夾時會自動讀的「說明書」，不用手動點開它 |
| `使用指南.md` | 圖文步驟，教你怎麼用 Claude Desktop 打開這個資料夾 |
| `常用句型.md` | 剪片時可以直接照抄的中文提示語範例 |
| `screenshots/` | 使用指南裡引用的截圖放這裡 |
